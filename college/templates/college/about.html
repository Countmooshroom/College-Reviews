About

Background

This Django web application was started by Ben Watts in Fall 2022 under the direction of Dr. Charlotte Gruner. He created the open source project on GitHub and welcomes others to contibute to the project. You can find the project here!

Goal

Provide COSC students with an opportunity to contribute to an ongoing project. Students who contribute to this project will learn:

  • HTML & CSS
  • Python
  • Django
  • GitHub

The Future

The project will eventually go live as a web application that will help Casper College students know what to expect before registering for classes.
